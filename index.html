Мова розмітки HTML

HTML- це мова для опису структури веб-сторінок. Сторінки, створені з його допомогою, можуть бути переглянуті тільки за допомогою спеціальних програм (браузерів), встановлених на комп'ютерах користувачів.
Абревіатура HTML розшифровується як Hyper Text Markup Language (мова розмітки гіпертексту).
Запам'ятайте, він не є мовою програмування, це мова розмітки.
HTML використовує теги розмітки, щоб описати структуру веб-сторінки.

Теги

HTML теги "- це ключові слова або символи, укладені в кутові дужки, наприклад" <body> "," <p>","<h2> "і т.д. Теги бувають двох видів: парні і одиночні (їх також називають порожніми). Парні теги складаються з відкриває і закриває тега, наприклад:"<p> "текст" </p>". Символ "/" після кутової дужки вказує на те, що тег закриває. Поодинокі теги складаються тільки з відкриваючого тега, наприклад:" <br>". Теги нечутливі до регістру, тому можуть бути написані як великими, так і малими літерами:" <P> "означає те ж саме, що і" <p>".
Теги визначають, де починається і де закінчується, HTML-елемент. За допомогою тегів браузер розпізнає структуру і кількість написаних вами тексту, наприклад, вони повідомляють браузеру, яка частина тексту є заголовком, де починається новий абзац, що потрібно підкреслити і де розташувати зображення (картинку). Отримавши цю інформацію, браузер використовує вбудовані в нього за умовчанням правила про те, як відображати кожен з цих елементів."

Що таке CSS?

  • CSS - Каскадні таблиці стилів (Cascading Style Sheets)
  • Стиль - правило, яке описує форматування окремого елемента на сторінці
  • Стилі були додані для вирішення проблеми оформлення веб-сторінок
  • Стилі можна зберігати в окремих документах, що зменшує розмір HTML-коду

Чим корисні таблиці стилів поміщені в окремий документ? Відповідь дуже проста, можна зібрати всі стилі, які використовуються на сайті, в один зовнішній файл з розширенням .css і зв'язати його з усіма сторінками сайту. Після цього, коли ви будете редагувати стиль,
зміни моментально зачіпають всі елементи на сторінках сайту, де є посилання на даний зовнішній файл зі стилями. Таким чином, ви можете повністю змінити зовнішній вигляд шляхом редагування єдиного файлу таблиці стилів, що істотно спрощує роботу, ніж редагувати стилі на кожній сторінці окремо.

JavaScript: Можливості та обмеження

JavaScript - це інтерпретована мова програмування, розроблений для взаємодії з веб-сторінками. JavaScript є реалізацією ECMAScript. ECMAScript - це просто опис мови, який визначений в стандарті ECMA-262.

JavaScript складається з трьох частин:

  • Ядро (ECMAScript) - базова функціональність JavaScript.
  • Об'єктна модель документа (Document Object Model, DOM) - кошти для роботи з вмістом веб-сторінок.
  • Об'єктна модель браузера (Browser Object Model, BOM) - кошти для взаємодії з браузером.
Mожливості Oбмеження
Додавати різні ефекти анімації JavaScript не може закривати вікна і вкладки, які не були відкриті з його допомогою
Реагувати на події - обробляти переміщення покажчика миші, натискання клавіш з клавіатури Не може захистити вихідний код сторінки і заборонити копіювання тексту або зображень зі сторінки
Здійснювати перевірку введення даних в поля форми до відправки на сервер, що в свою чергу знімає додаткове навантаження з сервера Не може здійснювати кроссдоменние запити, отримувати доступ до веб-сторінок, розташованим на іншому домені. Навіть коли сторінки з різних доменів відображаються в одне і теж час в різних вкладках браузера,
то код JavaScript належить одному домену не матиме доступу до інформації про веб-сторінці з іншого домену. Це гарантує безпеку приватної інформації, яка може бути відома власнику домену, сторінка якого відкрита в сусідній вкладці
Створювати і зчитувати cookie, витягувати дані про комп'ютер користувача визначати браузер Не має доступу до файлів, розташованим на комп'ютері користувача, і доступу за межі самої веб-сторінки, єдиним винятком є ​​файли cookie, це невеликі текстові файли, які JavaScript може записувати і зчитувати

Додаткова інформація

Довідник по HTML - це докладний посібник з мови розмітки HTML,
який включає всі оновлення і доповнення до мови в версії HTML 5.

для браузера користувача його структура завжди подається у вигляді HTML.

Довідник по CSS, відкриє перед вами практично безмежні можливості по оформленню інтернет сайту.
Перед вами постане захопливий світ різноманітності форм і кольорів, якими може володіти ваш сайт.

Вивчення JavaScript дасть вам можливість більш якісно задовольняти потреби відвідувачів вашого сайту.
